# Envoy Specification

## Overview
Envoy is an AI-powered economic intelligence platform.

## Agents
- **Consul:** Master Orchestrator & Interface. Manages UI and routes commands.
- **Sentinel:** Economic Intelligence Monitoring. Fetches RSS feeds (World Bank), summarizes with Gemini, and stores intelligence items.
- **Scribe:** Diplomatic Writer. Drafts reports using Gemini based on task instructions.
- **Attaché:** Delegation & Logistics Manager. (Prototype)
- **Connector:** Trade Matchmaking & CRM. (Prototype)
- **Sentinel-Legal:** Regulatory & Compliance Analyst. (Prototype)

## Tech Stack
- React, Node.js, SQLite, Gemini API.
